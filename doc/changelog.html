+ Added files missing from 1.3

Version 1.3

+ Module dumping has been added to dump.html
+ Dumps have been added in /dumps
+ Removed "unobstrusive-ajax.js" from html files to prevent 404 error

Notes

  • Size of dumps may be small for some modules, but they should include most of everything you would need for reversing anyway (code).
  • libSceWebKit2.sprx must be dumped in 17 parts because it's very large. Once you have all 17 parts, you can click "Stick WebKit", which will merge them. I tried to make the exploit dump the whole module at once, but it absolutely wrecked my PS4's ram
  • If you try to stitch WebKit without all 17 parts, the PHP page you are directed to will probably display errors and you may have to close the browser to get out. I plan to fix this in a future commit with proper error handling
  • Dumps are named .bin rather than .sprx to distinguish that they are dumped from memory and not by file, however they should still load fine in IDA for example
  • Big thanks to XorLoser, Maxton, and Fire30 for assistance with dumping modules
